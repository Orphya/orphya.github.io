# 世界观 {#world-views}

## 历法系统

本作拥有一套较为完整的历法系统，其对应如下：

| 月份 | 英文释义 | 中文释义 |
| :-: | :-: | :-: |
| 1 | Frostmoon | 霜月 |
| 2 | Halfwake | 半觉月 |
| 3 | Newbornmoon | 新生月 |
| 4 | Greenthief | 绿贼月 |
| 5 | Elvenslumber | 精灵垂眼月 |
| 6 | Aquaseek | 求水月 |
| 7 | Blazingsun | 烈阳月 |
| 8 | Stormstrike | 风暴月 |
| 9 | Wildreach | 荒元月 |
| 10 | Mortalstride | 人兵月 |
| 11 | Bloodmoon | 血月 |
| 12 | Soulbane | 残魂月 |

本作设定游戏第一日为 残魂月 22 日，即 12 月 22 日。
