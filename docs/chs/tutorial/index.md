## 基础

### 基本界面

![基本界面](/img/tutorial/8fa1cd5c98911955.png)

#### 体力

角色的血量，决定角色的生存能力。

#### 魔力

角色使用魔法的能力，身为精灵，魔力将随着自然环境而有所消长。

#### 贞洁点数

在某些特殊行为（例如卖淫）中，角色需要消耗贞节点数才能执行。角色在初始状态下每天仅获得 1 点贞节点数，并会在过夜后回复至满值。

#### 交互模式

角色当前的交互状态，分为和平模式、近战模式与远攻模式。

#### 快速按钮

快速打开对应面板，从左到右依次为[角色](#角色界面)、[背包](#背包界面)、[地图](#地图界面)、[任务](#任务界面)。

## 提示

::: info 行动前请三思！
如果对一个角色好，对方可能会因此喜欢妳 — 但也有可能他会利用妳，或觉得妳好操弄而欺负妳，因此在选择行为时务必谨慎。
:::

::: info 信任是基础。
当一个角色不够信任妳时，对方便不会轻易把私人的委托交给妳。试着为他做点什么，或者为他青睐的阵营付出来获取他们的信任吧！
:::

::: info 贞洁。
在某些特殊行为（例如卖淫）中，角色需要消耗贞节点数才能执行。角色在初始状态下每天仅获得 1 点贞节点数，并会在过夜后回复至满值。
:::

::: info 行动。
一些特定的任务有隐藏的触发条件，例如先收集了某项物品、或先完成了某个任务，有时甚至必须先过扎营过夜后才会触发...敬请尝试探索吧！
:::

::: info 有些委托每日只能执行一次！
虽然有些委托可以重复执行，但通常每天只能执行一次，妳必须在营火过上一夜才能重新接洽这些任务。
:::

::: info 他们（可能）来自不同阵营。
人物可能会因为自己身处的阵营有不同的立场，伤害他们的自己人将会招致怨恨，反之攻击敌对阵营则会得到他们的青睐 — 但要小心，从外貌上妳不知道他们对阵营的忠诚度。
:::

::: info 角色有自己的价值观！
不同的角色都有自己独特的价值观。例如，正义凛然的角色对于偷窃的行为非常不耻，而盗贼则对偷窃这个行为相对宽容。
:::
